# Portfólio Pessoal Minimalista ⬛⬜

[![Deploy Status](https://img.shields.io/badge/Deploy-Live-success?style=for-the-badge&logo=github)](https://zinrasalvino.github.io/)

Este é o repositório do meu portfólio pessoal. O projeto foi desenvolvido com foco em performance, acessibilidade e controle total sobre o estilo, adotando uma estética ultra-minimalista, monocromática e geométrica. 

A ideia central foi construir uma interface de alta fidelidade totalmente do zero, sem o uso de frameworks externos, demonstrando domínio sobre os fundamentos do front-end.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização responsiva, variáveis nativas (Custom Properties), CSS Grid, Flexbox e geometrias feitas puramente com código.
* **JavaScript (Vanilla):** Animações suaves de revelação (fade-in) baseadas no scroll do usuário (Intersection Observer API).

## ⚡ Funcionalidades e Seções

* **Design Responsivo:** Adaptável perfeitamente para desktops, tablets e smartphones.
* **Seção Sobre:** Um resumo da minha trajetória, unindo a base analítica da Física, a disciplina do esporte de alto rendimento e meu foco atual em Análise e Desenvolvimento de Sistemas.
* **Projetos:** Destaque para soluções full stack desenvolvidas, como o Bot de Mensagens via WhatsApp e o Sistema de Fila Virtual Universitária.
* **Contato Direto:** Links otimizados para comunicação rápida (E-mail, LinkedIn, GitHub e WhatsApp).
